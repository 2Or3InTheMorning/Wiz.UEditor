**Wiz.UEditor** 是一个基于 UEditor 构建的为知笔记编辑器插件。

#### 当前版本
2.0

#### 版本说明
基于 1.4.3 版本 UEditor

#### 主要修改
- 增加插入图片功能
- 增加屏幕截图功能
- 修复无法打开.md笔记
- 修复非中文界面无法安装
- 修复多次提示保存问题
- 修复Word图片转存问题

#### 下载安装
> 如果有原先的百度编辑器版本，请先卸载掉。

通过 [Github 下载](https://github.com/akof1314/Wiz.UEditor/releases "Github 下载")安装。

#### 使用方法
点击`编辑`按钮箭头，选择`百度编辑器`编辑。
